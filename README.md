<h1 align="center">OPENGATE</h1>
<h4 align="center">WiFi Hotspot Creator with Anti Tethering/Hotspot Bypass</h4>

### DESCRIPTION
OpenGate is a Linux based WiFi hotspot creation tool specifically engineered to bypass anti-tethering and hotspot restrictions imposed by carriers, ISPs, and network providers. It features intelligent client filtering capabilities and gives you complete granular control over how clients connect to your hotspot, allowing you to share your internet connection freely, seamlessly, and without detection.
